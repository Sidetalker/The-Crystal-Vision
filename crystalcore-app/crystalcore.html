CrystalCore

The open-source engine under Clementine: layered memory, semantic recall, profiles, and local models by default. Optional SpaceXAI chat only when you opt in. Memory files stay on your device.

What is real (built)

  • Sovereign companion runtime (Python package crystalcore)
  • JSON memory on disk you can open, edit, or delete
  • Local Ollama chat + embeddings by default
  • Terminal and localhost web UI (127.0.0.1 only)
  • Full transparency dump: python -m crystalcore.expose
  • Live status board: python -m crystalcore.status

What is design only

  • CrystalMatrix peer-to-peer mesh between companions
  • Planetary multi-node network
  • Zero-knowledge identity layer
